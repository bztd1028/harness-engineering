# 必备三方 Skill

1.首先进入项目根目录
2.根据以下命令完成项目所需 skill 的安装

## 基础能力
npx skills add vercel-labs/skills@find-skills -g -y
npx skills add anthropics/skills@skill-creator -y

## 软件研发综合能力包
npx skills add obra/superpowers -y

## API设计
npx skills add wshobson/agents@api-design-principle -y

## 前端开发
npx skills add anthropics/skills@frontend-design -y

## 评审
npx skills add affaan-m/everything-claude-code@security-review -y

## 测试
npx skills add anthropics/skills@webapp-testing -y

## 文档
npx skills add googleworkspace/cli@gws-docs -y

## Kubernetes
npx skills add jeffallan/claude-skills@kubernetes-specialist -y
